Référentiels
============

- Le dossier **RDF/** contient les référentiels des AN pour l'indexation des lieux.

- Le dossier **XML_SIA/** contient la version XML source pour les types de lieux concernés :
    - **FRAN_RI_005.xml** : référentiel des lieux hors Paris
    - **FRAN_RI_026.xml** : référentiel pour les édifices parisiens
    - **FRAN_RI_104.xml** : référentiel pour les statuts juridiques des collectivités
    - **FRAN_RI_011.xml** : référentiel pour les fonctions des collectivités
